## Текст раздела

В работе представлена методика валидации нейросетевых прогнозов ледовой обстановки, ориентированная на специфику задач навигационно-гидрографического обеспечения Северного морского пути.

**Основные результаты исследования:**

1. Показано, что традиционные метрики оценки качества прогнозов (RMSE, MAE, R²) не учитывают пространственное распределение ошибок и не позволяют оценить пригодность прогноза для навигационных задач.

2. Представлена интегрированная ледовая пограничная ошибка (IIEE) и её декомпозиция на компоненты AEE (ошибка площади) и ME (ошибка положения границы), позволяющая разделить два принципиально различных источника ошибок прогнозирования.

3. Обосновано, что для задач навигационно-гидрографического обеспечения минимизация ME имеет приоритет над минимизацией AEE, поскольку точное положение кромки льда критичнее общей площади ледового покрова.

4. Предложена четырёхэтапная методика валидации с критерием пригодности модели ME/IIEE < 0.5.

**Практические рекомендации:**

- При валидации моделей для НГО СМП использовать IIEE с декомпозицией на AEE и ME;
- Предпочтение отдавать моделям с минимальным ME при сопоставимых значениях IIEE;
- Анализировать соотношение OE/UE для выявления консервативных и оптимистичных прогнозов;
- Использовать независимые данные ДЗЗ для референсной валидации.

**Направления дальнейших исследований:**

- Разработка программного модуля для автоматизированного расчёта IIEE/AEE/ME и его интеграция в ГИС НГО;
- Исследование зависимости ME от горизонта прогнозирования для различных нейросетевых архитектур;
- Адаптация методики для российских источников данных ДЗЗ (Арктика-М, Метеор-М);
- Валидация методики на данных реальных навигационных операций по СМП.

Предложенная методика формирует основу для объективной оценки качества нейросетевых прогнозов в контексте их практического применения для обеспечения безопасности арктического судоходства.
